# TaskFlow - 과제 관리 앱

## 📋 프로젝트 소개

**TaskFlow**는 효율적으로 과제를 관리할 수 있는 웹 기반 애플리케이션입니다. 사용자는 간단하게 과제를 추가, 완료, 삭제할 수 있으며, 로컬 스토리지에 자동으로 저장됩니다.

## ✨ 주요 기능

- ✅ **과제 추가**: 새로운 과제를 입력하고 추가
- ✅ **과제 완료 표시**: 완료한 과제에 체크 표시
- ✅ **과제 삭제**: 더 이상 필요없는 과제 삭제
- ✅ **필터링**: 전체, 진행중, 완료된 과제별로 필터링
- ✅ **통계**: 전체, 진행중, 완료 과제 개수 표시
- ✅ **로컬 저장**: 브라우저를 닫아도 데이터 유지

## 🛠️ 기술 스택

- **HTML5**: 구조화된 마크업
- **CSS3**: 반응형 디자인
- **JavaScript (Vanilla)**: 모든 기능 구현
- **LocalStorage**: 데이터 저장

## 🚀 사용 방법

### 웹 브라우저에서 실행
1. 이 저장소를 클론하거나 다운로드합니다:
```bash
git clone https://github.com/dlaehrbs13-commits/2026131609-sw-.git
```

2. `index.html` 파일을 웹 브라우저에서 엽니다.

3. 과제를 입력하고 "추가" 버튼을 클릭합니다.

### GitHub Pages에서 실행 (온라인)
저장소 설정에서 GitHub Pages를 활성화하면, 다음 URL에서 접속할 수 있습니다:
```
https://dlaehrbs13-commits.github.io/2026131609-sw-/
```

## 📖 사용 가이드

### 과제 추가
1. 입력 필드에 과제를 입력합니다.
2. "추가" 버튼을 클릭하거나 Enter 키를 누릅니다.

### 과제 완료 표시
- 과제 항목의 체크박스를 클릭하면 완료 표시됩니다.

### 과제 삭제
- 과제 항목의 "삭제" 버튼을 클릭합니다.

### 필터링
- 상단의 필터 버튼(전체, 진행중, 완료)을 클릭하여 과제를 필터링합니다.
